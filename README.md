# 📊 Survey System v3.1
Консольно-графическое приложение для сбора, хранения и анализа данных опросов с интеграцией внешних справочников.

## 🚀 Возможности
- ✅ GUI на Tkinter с вкладками (Опрос, Аналитика, Экспорт)
- ✅ Хранение в SQLite с автоматической датой и временем
- ✅ Импорт "грязных" JSON-справочников с автоматической очисткой ключей
- ✅ SQL-аналитика: группировка, подсчёт, сортировка, объединение таблиц
- ✅ Экспорт результатов в CSV для Excel
- ✅ Подсказки городов из базы данных

## 🛠️ Запуск
1. Убедитесь, что установлен Python 3.10+
2. Клонируйте репозиторий или распакуйте папку
3. Запустите: `python run.py`
4. Данные сохраняются в `data/survey.db`

## 📂 Структура
- `src/` — исходный код приложения
- `scripts/` — утилиты обработки данных
- `data/` — файлы данных (JSON, DB, CSV)
- `run.py` — точка входа
